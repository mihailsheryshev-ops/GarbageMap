# GarbageMap — Сборщик мусора

Мобильное приложение для Android — отмечай места скопления мусора на карте. Метки видят все пользователи в реальном времени.

## Описание
Нажми на любое место на карте → появится диалог → подтверди → метка сохранится в Firebase и появится у всех пользователей. Вкладка «Список» показывает все метки с возможностью удаления.

## Требования
- Android 7.0 и выше (API 24+)
- Интернет

## Стек технологий
- Kotlin
- Яндекс MapKit
- Firebase Realtime Database
